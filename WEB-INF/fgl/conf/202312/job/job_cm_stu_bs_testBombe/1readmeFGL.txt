FGL 20250313:
Auftrag mit Libre-Office ODT Datei.
Dieser ist so aufgebaut, dass mit Libre-Office <= 7.6 zumindest 
der Libre Office Server crasht.

Neuer Test mit Libre-Office 24.8.5.2. ergibt aber, dass der Server diese ODT Datei verarbeiten kann.

Merke:
Man kann aber den Prozess so gestalten, dass der Server wieder neu startet wenn er mal crasht, 
bzw. Hisinone versucht ggfs. selbst den Server neu zu starten.
